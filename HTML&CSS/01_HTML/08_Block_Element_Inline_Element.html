div태그와 span태그

  • 둘 다 페이지의 영역을 나누는 역할
  • Sementic하지는 않다
  • div 태그는 다른 태그들을 담는 컨테이너 역할을 하기도 한다
  • span 태그 또한 마찬가지이지만, 컨테이너 역할을 하지는 않음

실습

4절에 해당하는 부분이 1897년 노래의 본절이고 나머지 1~3절은 윤치호가 추후 작사했다.
해당 시기를 다룬 1959년 영화 독립협회와 청년 리승만에서도 독립문 정초식에서 사람들이 무궁화 노래를 부르는 장면이 나온다.
  • div태그는 박스 전체에 백그라운드 컬러가 적용, span 태그는 컨텐츠 만큼만 적용
  • div 태그는 영역, 컨테이너. block 레벨 요소
  • span 태그는 영역, 특정 아이템. inline 레벨 요소
  • 모든 html 태그는 초기 디스플레이 설정값을 갖는다. 이게 block이냐 inline이냐에 따라 달라짐.

    • Block Element

  • 블록 레벨 요소는 부모 요소의 전체 공간을 차지하여 "블록"을 만든다. h, ol, ul, li, p 태그 등이 여기에 속한다
  • 화면 구성이나 레이아웃을 짤 때는 블록 레벨 요소를 사용한다
  • 블록 레벨 요소는 한 칸을 모두 차지하기 때문에 세로로 나열된다
  • width, height, margin 속성이 적용됨

    • Inline Element

  • 인라인 레벨 요소는 컨텐츠의 흐름을 끊지 않고(줄바꿈x), 요소를 구성하는 태그에 할당된 공간만 차지. a, em, img, span 태그가 여기에 속한다.
  • 인라인 레벨 요소는 컨텐츠 영역 만큼 차지하기 때문에 가로로 나열
  • margin-top, margin-bottom이 적용되지 않음. 대신에 line-height 사용
  • width, height 속성이 적용되지 않음
  • 태그가 컨텐츠 할당 곤강만 갖고 있기 때문에 text-align과 같은 속성은 사용할 수 없음.
