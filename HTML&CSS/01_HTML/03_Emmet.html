• ! + enter = 문서 양식 자동 완성
  • div>ul>li + tab = 자식 노드 자동 완성
  • div>ul+ol+div = 형제 노드 자동 완성
  • div>ul>li*3 = 동일 태그 반복 생성
  • span#item = 아이디(item)를 갖는 태그 생성
  • #item = 태그의 기본 값은 div이기 때문에 자동으로 item 아이디를 갖는 div 생성
  • span.title = class를 갖는 태그 생성
  • .title = 태그의 기본 값은 div이기 때문에 자동으로 class를 갖는 div 생성
  • p.container{Hello World~!} = 클래스와 컨텐츠가 입력된 태그 생성
  • p.container{item$}*5 = 넘버링 부여
