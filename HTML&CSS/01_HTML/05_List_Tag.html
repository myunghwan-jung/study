목록 태그

  • <ol> : Oldered List. 순서가 있는 목록을 표현할 때 사용. type속성으로 글머리 기호를 변경 가능
  • <ul> : Unordered Lists. 순서가 없는 목록을 표현할 때 사용
  • <li> : Listed Item. 목록 하위 항목으로 사용. <ol>, <ul> 태그의 하위에 위치
  • <dl> : Definition List. 정의 목록. 사전처럼 용어를 설명하는 목록 생성.
    EX: A는 B이다 같은 Key=Value로 사용할 떄 유용
  • <dt> : Definition Term. 정의 용어. 정의되는 용어의 제목을 넣을 때 사용.
  • <dd> : Definition Description. 정의 설명. 용어를 설명하는 데 사용.

주의 사항

  • <dl> 태그는 하나 이상의 <dt>와 <dd>쌍의 태그를 갖고 있어야 한다.
    단, 태그가 반드시 하나의 짝으로 지어져야 하는 것은 아니다.
  • <li>,<dt>=<dd>태그는 밖에서 독립적으로 사용할 수 없다.
  • <ul> 태그 하위 요소로는 <li> 태그가 위치해야 한다.
