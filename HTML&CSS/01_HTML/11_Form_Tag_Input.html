HTML Form 다루기 1

  • HTML에서 Form은 웹에서 사용자의 정보를 입력받기 위해 사용함
  • 로그인, 회원가입, 게시판 글쓰기 등 데이터를 입력받을 때, 각 데이터를 폼 데이터 또는 필드라고 부르고, 이러한 데이터들의 묶음을 폼이라고 부른다
  • Form은 유저의 정보를 입력받을 수 있게 만들어 놓은 양식
  • <form> 요소는 정보를 제출하기 위하여 어디서부터 어디까지가 양식인지 지정하는 역할
  • action : 양식 데이터를 처리할 서버 프로그램의 URI
  • method : 양식을 제출할 때 사용할 HTTP 메서드
  • post : 양식 데이터를 요청 본문으로 전송
  • get : 양식 데이터를 URL의 쿼리스트링으로 붙여서 전송

    Input 태그

  • <input> 요소로 데이터를 입력 받을 수 있다. type 속성을 통해 다양한 방법으로 데이터를 받을 수 있음

    • text

  • text : input 태그의 기본값. 한줄의 텍스트를 입력받는다
  • email : email 데이터를 받을 때 사용. 이메일 유효성 검증
  • tel : 전화번호를 받을 때 사용. 모바일 접근 시 키패드가 다름

    • label

  • label 레이블 태그는 입력받는 필드를 설명할 때 사용
    사용 방법 : label 태그 하위에 input 태그를 위치, id와 for 속성을 사용하여 input 태그와 연결
