Form 데이터 태그 속성

  • required : 입력값이 필수라는 것을 명시
  • readonly : 필드를 읽기 전용으로 만든다. 데이터를 수정할 수는 없지만 서버로 전송됨
  • disabled : 비활성화 시킬 수 있으며 해당 필드는 서버로 전송되지 않음
  • autofocus : 초기에 해당 필드에 커서를 위치 시킴
  • placeholder : 입력 필드가 비어있을 때 해당 입력값의 설명 또는 가이드 문구를 삽입

Form 데이터 태그 속성 테스트

Checkbox

여러개의 체크박스 항목 중 2개 이상 선택할 수 있음.
그리고 체크박스 선택 시 선택된 체크박스의 value값이 서버로 전송

Radio

여러개의 라디오 항목 중 1개를 선택
그리고 라디오 항목 선택 시 선택된 항목의 value 값이 서버로 전송
여러개 중 하나를 선택하게 하려면 항목들의 <radio name=""> name 속성 값을 같은 값으로 그루핑 헤줘야 함

Select

<select> 태그는 옵션 메뉴를 제공. <option> 태그로 각 항목을 나타내며, <select> 태그는 <option> 태그를 감싼다
옵션 안에 있는 내용은 사용자 화면에 보여주는 내용이며 실제 서버로 전송되는 값은 value 값이다
  • selected 해당 속성은 현재 <select> 요소에서 선택된 <option> 을 가르킨다

    Button

    <button> 태그는 클릭 가능한 버튼을 나타낸다
    <form> 내부는 물론이고 버튼 기능이 필요한 곳 어디나 배치 가능

    type

    버튼의 행동 방식을 선언
  • submit : 버튼이 서버로 양식 데이터를 제출(기본값)
  • reset : <input type="reset"> 처럼, 모든 입력 필드를 초기값으로 되돌림
  • button : 기본 행동이 없으며, 주로 클릭 후 자바스크립트 측 코드를 명령할 때 사용
