Box Model

모든 HTML 요소는 웹 페이지에서 일정 공간을 차지한다. 이러한 공간을 박스 모델이라고 정의한다
HTML 요소의 박스 모델은 Content, Padding, Margin으로 구성되어있다
  • Content - 텍스트나 이미지가 들어있는 HTML 요소의 실질적인 내용
  • Padding - Content와 Border 사이의 영역으로 쉽게 말해 안쪽 여백
  • Border - Content를 감싸는 테두리
  • Margin - 테두리와 이웃하는 요소 사이의 간격. 바깥쪽 여백

    • Box Model CSS 속성

    박스 모델은 속성을 지정하지 않으면 브라우저가 선언한 기본값으로 세팅, CSS 속성으로 박스모델 값을 변경할 수 있음

    Content

    • width : Content 영역은 width 값으로 가로 너비를 지정
    • height : Content 영역은 height 값으로 세로 너비를 지정
    • 인라인 레벨 요소에는 width, height가 적용되지 않음. 인라인 요소는 컨텐츠 만큼의 영역을 갖기 때문
      만약 인라인 요소의 width, height를 변경하고 싶다면 display: inline-block으로 변경해야 함
      inline-block : block과 inline의 중간 형태. 요소는 inline인데 내부는 block처럼 표시

    바깥 여백 Margin

    • margin 속성은 HTML 요소의 바깥 여백을 지정한다
    • margin-top, margin-bottom, margin-left, margin-right 속성을 사용하여 각각 지정 가능

    마진 중첩

    HTML 요소를 세로로 배치할 경우 margin과 margin이 만날 때 margin 값이 큰 쪽으로 겹쳐지는 것
    요소를 가로로 배치할 경우에는 상관 없지만 세로로 배치할 경우에는 값이 큰 margin만 적용된다

    안쪽 여백 Padding

    • padding 속성은 HTML 요소의 안쪽 여백을 지정한다
    • padding-top, padding-bottom, padding-left, padding-right 속성을 사용하여 각각 지정하는 것도 가능
