글자 색상

글자 색상은 color 속성으로 지정하며 색상 값으로는 16진수 값, RGB 값, HSL 값, 색상 이름이 사용된다

색상 키워드 표기법

색상 키워드는 대소문자를 구분하지 않는 식별자. red, blue, black 등 특정 색을 나타낸다
이름이 표현하는 색을 지칭하지만, 모든 키워드의 본질은 인위적이며 어떤 규칙을 따르지 않는다
  • 예시
    blue, red, yellow 등 색상 키워드
  • tranparent 키워드
    transparent 키워드는 완전히 투명한 색. 색을 입힌 항목의 뒷편이 모두 보인다
    기술적으로 transparent는 rgba(0,0,0,0)의 짧은 이름이다
  • currentColor 키워드
    currentColor 키워드는 요소의 color 속성값을 나타낸다. 이를 통해 다른 속성이 color 속성값을 따라가도록 설정할 수 있다

RGB/RGBA 표기법

RGB 색상 모델은 빨강, 초록, 파랑을 통해 특정 색을 표현한다. 선택사항으로 색의 투명도를 알파 채널로 표현할 수 있다
함수 표기법 rgb(), rgba()로 표현
  • 예시
    rgb(0,0,0),rgba(0,0,0,0)

16진수 표기법

RGB 색상 모델은 빨강, 초록, 파랑을 통해 특정 색을 표현한다. 선택사항으로 색의 투명도를 알파 채널로 표현할 수 있다
  • 예시
    #ff0000,#ff00ff,#808000 등 16진수

hsl/hsla 표기법

HSL 색상 모델은 색상, 채도, 명도를 통해 특정 색상을 표현한다. 선택사항으로 색의 투명도를 알파 채널로 표현할 수 있다
HSL 색상은 함수명 hsl()와 hsla()를 사용한다
  • 예시
    hsl(H, S,L[,A]), hsla(H, S, L, A)
