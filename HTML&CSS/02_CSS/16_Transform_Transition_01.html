Transform

transform 속성은 HTML 요소를 회전, 크기 조절, 기울이기, 이동 효과를 나타낼 때 사용
사용법은 transform 속성 값으로 특수 함수를 넣어주면 됨
주의사항 : transform을 사용하려면 해당 요소의 display 속성이 block 또는 inline-block이어야 함

transform 함수

  • translate(tx, ty)
    지정한 크기만큼 x축, y축으로 이동
  • translateX(tx)
    지정한 크기만큼 x축으로 이동
  • translateY(ty)
    지정한 크기만큼 y축으로 이동
  • scale(sx, sy)
    지정한 크기만큼 x축과 y축으로 확대, 축소
  • scaleX(sx)
    지정한 크기만큼 x축으로 확대, 축소
  • sacleY(sy)
    지정한 크기만큼 y축으로 확대, 축소
  • rotate(각도)
    지정한 각도만큼 회전(+ 시계방향, - 시계반대방향)
  • rotateX(각도)
    x축을 기준으로 회전. 이 때 입체감있게 표현하려면 perspective 속석을 부모 요소에 적용해야 함
  • rotateY(각도)
    y축을 기준으로 회전. 이 때 입체감있게 표현하려면 perspective 속석을 부모 요소에 적용해야 함
  • rotateZ(각도)
    z축을 기준으로 회전. 이 때 입체감있게 표현하려면 perspective 속석을 부모 요소에 적용해야 함
  • skew(ax, ay)
    지정한 각도만큼 x축과 y축으로 왜곡
  • skewX(ax)
    지정한 각도만큼 x축으로 왜곡
  • skewY(ay)
    지정한 각도만큼 y축으로 왜곡

Transition

transition의 사전적 의미는 전환. CSS에서 transition은 속성 값이 변할 때 더욱 부드럽게 전환할 수 있도록 도와줌
예를 들어 100x100 이미지에 커서를 올리면 200x200으로 변환하다고 했을 때 부드럽게 바뀌게 만드는 것

Transition 속성

  • transition-property
    어떤 속성에 트랜지션 효과를 줄 것인지 지정
    • transition-property : 속성1, 속성2;와 같이 지정
    • all : 모든 속성을 지정(생략 가능)
    • none : 아무것도 지정하지 않음
  • transition-duration
    트랜지션 효과를 몇 초 동안 실행할 것인지 지정
  • transition-delay
    지정한 초 만큼 기다렸다가 실행
  • transition-timing-function
    트랜지션이 시작하면서 끝날때의 타이밍(속도 지정)
    • linear : 트랜지션의 시작과 끝의 속도가 일정
    • ease-in : 천천히 시작했다가 완료될 때 속도가 증가
    • ease-out : 빨리 시작했다가 완료될 때 속도가 감소
  • transition
    transition 단축 속성으로 위 속성들을 한번에 표기
