독일의 前 축구 선수이자 튀르키예의 정치인.
축구 선수 시절 포지션은 공격형 미드필더.
현재 튀르키예의 여당인 정의개발당의 중앙협의회 위원을 역임하고 있다.
천재적인 패싱 능력을 보유했던 선수로서,
출전한 모든 주요 대회[5]에서 도움왕을 차지한,
명실상부한 2010년대 아스날의 슈퍼스타였으며,
국가대표팀에서도 2014 월드컵 우승을 이끈 주역이다.

Inline Style 실습

꾸밀 태그 안에 바로 적용하는 것이 인라인 스타일
세미콜론을 통해 여러 스타일을 적용 가능
실전에서는 스타일이 절대 바뀌지 않도록 개별 적용해야 하는 경우에 사용한다

Internal Style 실습

Internal Style은 head 태그 안에 style 태그를 선언하여 꾸미는 방법
실전에서는 잘 사용하지 않는다. 다른 파일에서도 동일한 스타일을 적용하려면 불편하기 때문

External Style 실습

외부에 style.css 파일을 생성, 스타일을 적용한 후 link 태그로 html에 불러온다
관리하기도 편하고 용량도 작기 때문에 가장 자주 사용되는 방법
