테두리 border

CSS border border-width border-style border-color 속성으로 요소의 테두리를 설정

    border-style

  • 어떤 형태의 테두리 스타일을 지정할지 나타냄
  • 4개의 방향의 값을 한번에 지정할 때는 방향 순서를 지켜야 함. top right bottom left
  • 테두리 스타일 종류는 MDN 참고

    border-width

  • 테두리 두께를 지정
  • 값으로 수치 또는 키워드 thin medium thick

    border-color

  • 테두리 색상을 지정

    border

  • 단축 속성. 다음의 하위 속성을 포함 : border-width, border-style, border-color
  • 테두리 두꼐, 스타일, 색상을 한번에 표기
  • border-top, border-right, border-left, border-bottom을 사용하여 스타일 개별 지정 가능
  • 예시 : border: 1px solid red

    border-radius

  • 테두리 꼭지점을 둥글게 만든다

Box Sizing

Box Sizing 속성은 HTML 요소의 너비와 높이를 계산하는 방법을 지정. CSS 박스 모델에서 지정한 너비와 높이는 HTML 요소의 Content 크기에 적용된다
요소에 테두리나 안쪽 여백이 있으면 너비와 높이에 더해서 화면에 그려진다. 따라서 크기를 설정할 때 원하는 크기를 얻으려면 테두리 및 안쪽 여백을 고려해야 한다
이 떄, box-sizing 속성을 사용해 이 방식을 변경할 수 있다

    box-sizing 값

  • content-box : 기본 CSS 박스 크기 결정법. 요소의 너비를 100px로 설정하면 컨텐츠 영역이 100px 너비를 가지고, 테두리와 여백은 이에 더해진다
  • border-box : 테두리와 안쪽 여백도 요소의 크기에 포함시킨다.
    너비를 100px로 설정하고 테두리와 안쪽 여백을 추가하면 컨텐츠 영역이 줄어들어 총 너비 100px를 유지한다
    대부분의 경우 이 편이 크기 조절에 용이하다
