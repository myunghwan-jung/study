Animation

요소에 적용되는 CSS 스타일을 다른 CSS 스타일로 부드럽게 전환시켜 주는 것
이 떄 CSS 스타일이 변화되는 중간 지점을 키프레임이라고 한다. 즉, @keyframes는 애니메이션의 중간 상태
애니메이션은 트랜지션보다 훨씬 더 규모가 크고 복잡하며 다양한 능력을 가지고 있기 때문에 좀 더 정밀한 효과를 구현 가능

keyframes 정의하기

@keyframes로 애니메이션을 생성할 수 있다
  • from(시작)~to(끝)을 이용한 애니메이션 shape 생성
  • from ~ {percent} ~ 를 이용한 애니메이션 background 생성

Animation 속성

  • animation-name
    애니메이션의 중간 상태를 지정하기 위한 이름을 정의. 중간 상태는 @keyframes 규칙을 이용하여 기술
  • animation-duration
    한 싸이클의 애니메이션이 얼마에 걸쳐 일어날 지 지정
  • animation-delay
    엘리먼트가 로드되고 나서 언제 애니메이션이 시작될지 지정
  • animation-iteration-count
    애니메이션이 몇 번 반복될지 지정. infinite로 지정하면 무한히 반복
  • animation-play-state
    애니메이션을 멈추거나 다시 시작할 수 있음
  • animation-timing-function
    중간 상태들의 전환을 어떤 시간 간격으로 진행할지 지정
  • animation-fill-mode
    애니메이션이 시작되기 전이나 끝나고 난 후 어떤 값이 적용될지 지정
  • animation
    animation들의 단일 속성으로 한꺼번에 작성
