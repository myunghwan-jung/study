가상 클래스

가상 클래스(pseudo-class)는 선택하고자 하는 HTML 요소의 특별한 상태(state)를 명시할 때 사용
  • 문법
    선택자: 가상클래스이름 {속성: 속성값;}
  • 예제
    a:hover { color: orange; } input:focus { color: red; }
  • 대표적인 css 가상 클래스
    • :link : 아직 방문하지 않은 요소. href속성을 가진 a, area, link 중 방문하지 않은 모든 요소를 선택
    • :visited : 사용자가 방문한 적이 있는 링크
    • :active : 사용자가 활성화한 요소(버튼 등)을 나타냄
    • :hover : 사용자의 마우스 포인터가 요소 위에 올라가 있으면 선택
    • :focus : 양식의 입력 칸 등 포커스를 받은 요소를 나타냄. 보통 사용자가 요소를 클릭 또는 탭하거나 키보드 tab으로 선택했을 때 발동
    • :nth-child : 형제 사이에서의 순서에 따라 요소를 선택
    • :not(selector) : not(selector) 안에 포함된 요소를 제외
    대표적인 가상 클래스를 사용할 때는 link -> visited -> hover -> active 순서로 선언하는 것을 권장
    순서가 달라지면 적용이 안될 가능성이 있기 때문

가상 요소(Psuedo element)

가상 요소는 해당 HTML 요소의 특정 부분만을 선택할 때 사용
  • 문법
    선택자::가상요소이름 {속성: 속성값;}
  • 예시
    p::first-line { color: blue; text-transfrom: uppercase; }
  • 대표적인 CSS 가상 요소
    • ::first-letter : 텍스트의 첫 글자만을 선택. 블록 레벨 요소에만 사용 가능
    • ::first-line : 텍스트의 첫 라인만 선택. 블록 레벨 요소에만 사용 가능
    • ::befor : 특정 요소의 내용(content) 부분 바로 앞에 다른 요소를 삽입할 때 사용
    • ::selection : 해당 요소에서 사용자가 선택한 부분만을 선택할 때 사용
